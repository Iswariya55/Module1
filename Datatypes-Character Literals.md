# EXP 3: Datatypes-Character Literal in Python

##  Aim
To write a Python program that prints the characters `'V'` and `'C'` using character literals.

## Algorithm
1. Print the character `'V'`.
2. Print the character `'C'`.

## Program
```
v='V'
w="c"
print(v)
print(w)
```
